## 📖 About The Project

Quiz Lab is a modern trivia and knowledge-testing platform designed to deliver an engaging quiz experience. Users can select different game modes, explore multiple categories, and enjoy a smooth interface optimized for desktop, tablet, and mobile devices.

### Key Features

* Multiple Quiz Modes
* Category-Based Questions
* Responsive Design
* Offline Support
* Interactive UI & Animations
* Fast Performance with Next.js
* Mobile Friendly Experience

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/aayush8797823674/Quiz-Game.git
```

Move into the project directory:

```bash
cd Quiz-Game/source_code
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```
